# A LaTeX version of Raphael Kühner, Bernhard Gerth, _Ausführliche Grammatik der griechischen Sprache_

This is a side project to edit and annotate the great Raphael Kühners,
Bernhard Gerth and Friedrich Blass grammar _Ausführliche Grammatik der
griechischen Sprache_ using LaTeX and its tools.

The project is free and open source and I do not guarantee any thing (neither
about quality neither about timeline).

I have been partially using the edition available at
[Perseus](https://www.perseus.tufts.edu/hopper) and [PerseusUnderPhilologic](https://perseus.uchicago.edu/index.html),
by means of the good and old ctrl-c ctrl-v as they do not make the source code available.

I hope I document this mess soon enough.
